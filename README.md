# ♟ Ход дрона — Поиск по регламенту

Сайт для поиска информации по регламенту соревнования **«Ход дрона. Шахматные сражения гетерогенных роёв дронов»** (Москва, Сколково, 2026) с помощью LLM-агента.

🌐 **[gebondar.github.io/Hod_Drone_Reglament](https://gebondar.github.io/Hod_Drone_Reglament)**

## Возможности

- **💬 LLM-поиск** — задайте вопрос по регламенту, модель найдёт ответ со ссылками на разделы
- **📋 Просмотр регламента** — полный текст с оглавлением и навигацией по разделам
- **13 бесплатных моделей** — OpenAI GPT-OSS 20B (по умолчанию), DeepSeek Chat V3, Nemotron 3 Super 120B и другие через OpenRouter
- **Адаптивность** — работает на телефонах и десктопе

## Как работает

1. Сайт загружает актуальный `reglament_v02.md` из репозитория
2. При поиске полный текст регламента отправляется как **system prompt** в OpenRouter API
3. LLM-модель отвечает, опираясь исключительно на текст регламента, со ссылками на конкретные пункты
4. API-ключ хранится в **GitHub Secrets** и подставляется при деплое через GitHub Actions — в исходниках его нет

## Технологии

- Статический HTML/CSS/JS (GitHub Pages)
- [OpenRouter API](https://openrouter.ai/) — доступ к бесплатным LLM
- [marked.js](https://marked.js.org/) — рендеринг Markdown (регламент + ответы)
- GitHub Actions — безопасный деплой с инжектом API-ключа

## Разработка

```bash
# Локальный просмотр
open docs/index.html

# Деплой
git push origin main  # GitHub Actions соберёт и задеплоит
```

### Добавление моделей

Отредактируйте `<select id="modelSelect">` в `docs/index.html`. Только бесплатные модели с `:free` в ID или с нулевой ценой на OpenRouter.

### Обновление регламента

Отредактируйте `reglament_v02.md` — сайт подгружает его динамически. Копия в `docs/reglament_v02.md` нужна для GitHub Pages.
